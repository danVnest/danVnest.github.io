    This website is still in development, you can see all the code here. You can see the historical development process here

    This website is designed to be a little bit unique and unconventional - so as to be memorable, but also to be as minimal and clear as possible to ensure usability.

    You can see that its purpose is to introduce you to who I am, and what I can do. I decided not to just list my "skills" and "experience" as you would see on a typical resume (which is somewhat subjective), instead you can find relevent projects and judge for yourself!

    I decided to make this website mostly just to learn how to do so. I had some previous experience from prototyping iQ Consult's website and backend interface, but wanted to develop this site from scratch with minimal framework assistance as a programming exercise.

    If you want to learn a bit more about what makes a website: HTML5 dictates the sites content, CSS3 the style and strucutre, jQuery the responsiveness and interactability. Follow those links to see some great little introductions to each concept.

    Another thing worth noting is that the CSS3 code is compiled through LESS, which gives the developer the ability to program styles more dynamically, requiring much less code duplication than pure CSS.
